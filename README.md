## 🛠️ Installation (Developer Mode)

Since this is a custom premium tool, follow these 30-second steps to install it:

1. **Download the Code**: Click the green **Code** button above and select **Download ZIP**, then extract it.
2. **Open Extensions**: In Chrome, go to `chrome://extensions/`.
3. **Developer Mode**: Toggle **Developer mode** in the top-right corner to **ON**.
4. **Load Extension**: Click **Load unpacked** and select the `ai-limit-tracker` folder from this repository.
5. **Pin it**: Click the 🧩 icon in your toolbar and **Pin** AI Limit Tracker for instant access.
